=====
Fun-Olympic
=====

Fun olympic is a Django app to conduct web-based service for users. Admin can manage site,
visitors can view site.


Quick start
-----------

1.  For this project, python virtual environments is required.
        Here is link for hot to create it.

        link:
        https://www.freecodecamp.org/news/how-to-setup-virtual-environments-in-python/

2.  For project setup there is requirements.txt file and all the packages 
    can be installed form there.


2. Start the development server by python manage.py runserver 7000
and visit http://127.0.0.1:8000/admin/login/ for Admin site.
